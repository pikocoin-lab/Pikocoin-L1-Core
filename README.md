# 🪙 Pikocoin (PIKO) - 量子时代的 L1 区块链实验

Pikocoin 是一个基于 Modal 云端原生构建的轻量级 Layer 1 区块链原型，旨在探索 AI 算力证明与抗量子加密的可能性。

### 🚀 项目亮点
- **云端大脑：** 利用 Modal 的 Serverless 技术处理分布式账本。
- **全民挖矿：** 即使是老旧设备（如我的 i5-4590），也能作为算力节点参与。
- **抗量子预留：** 核心逻辑已集成加密库，未来将升级至 ML-DSA 签名算法。
